# TBH MarketLens

**_TBH: Task Bar Hero_ のアイテムにカーソルを合わせてキーを押すだけで、Steam マーケットの現在価格が即表示。名前を打って検索する手間はありません。**

by **Ghost Shark Robotics** · UI は 日本語 / English / 中文 対応

---

## ⬇ ダウンロードと起動（Windows）

1. **[📥 Releases ページ](https://github.com/GhostSharkRobotics/tbh-marketlens/releases)** を開く
2. 一番上（最新版）の **「Assets」** の中の、**`TBH-MarketLens` で始まる `.zip` ファイル**をクリックしてダウンロード
3. ダウンロードした zip を **右クリック → すべて展開**（ファイルはフォルダごとまとめて置く）
4. **`TBH MarketLens.exe`** をダブルクリックで起動 → **タスクトレイ**（時計の近く）に常駐。初回に使い方が出ます

> 💡 「**WindowsによってPCが保護されました**」が出たら **詳細情報 → 実行**（署名なしアプリのための表示で問題ありません）。ホットキー系アプリは PyInstaller の都合でウイルス対策が誤検知することがあります。信頼できる場合は許可してください。

---

## できること

- **アイテムにカーソル → 発動キー → カードで Steam 市場価格**（最安・中央値）を表示。押した時に取得。
- カーソル直下のアイテムを**画面OCR**で読み、ゲームのアイテムDBと照合（装備・宝石・彫刻・刻印・素材ほか）。名前・レア度・種別は **日本語 / English / 中文**。
- **価格履歴**（トレイ）：追加日/最安/中央値で並べ替え、お気に入り、名前・レア度の修正、削除、**一括更新**。セッションをまたいで保存。
- **出品待ち**（トレイ）：Steam 在庫の出品ホールドを追跡し、出品可になったら通知。
- 発動キーは**任意のキー/組み合わせ**に変更可（既定：マウスの**戻る/サイド**ボタン）。**Windowsと一緒に起動**も選べます。

## 安全？（チート対策）

はい。MarketLens は**画面に映っているものを読むだけ**です。Windows 標準の画面キャプチャ（スクリーンショットと同じ）で画面を撮り、OCR でアイテム名を認識して、その価格を Steam で調べます。別プロセスで動き、**ゲームのメモリを読んだり書き換えたりしません**。だからゲームのアンチチートに**チートとして検出されません**。

## プライバシー

匿名の利用統計（起動・参照したアイテム名・エラー）だけをランダムIDで送ります。**IP・Steam在庫・個人情報は一切送りません**。**設定**でいつでもオフにできます。

## 更新

起動時に新版を確認。あれば**トレイ/設定に「⬆ 更新」**が出ます。**1クリックでDL→差し替え→自動再起動**（手動の再DL・再展開は不要）。
