# TBH MarketLens

**在 _TBH: Task Bar Hero_ 中把光标对准物品并按一个键，立刻弹出它的 Steam 市场价格 —— 无需打字、无需 Alt+Tab。**

by **Ghost Shark Robotics** · 界面支持 日本語 / English / 中文

---

## ⬇ 下载与运行（Windows）

1. 打开 **[📥 Releases 页面](https://github.com/GhostSharkRobotics/tbh-marketlens/releases)**
2. 在最新版本的 **“Assets”** 中，点击以 **`TBH-MarketLens`** 开头的 **`.zip`** 文件下载
3. **右键 zip → 全部解压**（请把文件保持在同一个文件夹里）
4. 双击 **`TBH MarketLens.exe`** 运行 → 它常驻在**系统托盘**（时钟附近）。首次运行会显示用法

> 💡 若出现 **「Windows 已保护你的电脑」**，点 **更多信息 → 仍要运行**（程序未签名，属正常现象）。用 PyInstaller 打包的快捷键类程序，杀软可能误报；信任来源即可放行。

## 功能

- **对准物品 → 按快捷键 → 卡片显示 Steam 市场价格**（最低价 + 中位价），按下时实时获取。
- 通过**屏幕 OCR** 识别光标正下方的物品，并与游戏物品数据库匹配（装备、宝石、雕刻、铭文、素材等）。名称、品质、类型支持 **日本語 / English / 中文**。
- **价格历史**（托盘）：按添加时间/最低价/中位价排序、收藏、重命名/纠正识别、改品质、删除、**一键全部更新**，并跨会话保存。
- **可出售计时**（托盘）：追踪 Steam 库存的上架冷却，物品可出售时通知你。
- 触发键可改为**任意按键/组合**（默认：鼠标**后退/侧**键）。可选**随 Windows 启动**。

## 安全吗？（反作弊）

是的。MarketLens 是**完全独立的程序**，只做三件事：①截取自己/桌面画面并识别文字，②监听你设定的快捷键，③向 **Steam 公开价格 API** 查询价格。它**从不读写游戏内存、不注入任何东西、完全不接触游戏进程**（也无加速/时间操作），因此不会触发游戏的反作弊。

## 隐私

仅发送匿名使用统计（启动、查询的物品名、错误），并绑定到一个随机 ID —— **绝不发送你的 IP、Steam 库存或任何个人信息**。可在**设置**中随时关闭。

## 更新

启动时自动检查。有新版本时，**托盘/设置中会出现「⬆ 更新」** —— **一键下载、替换文件并自动重启**，无需手动重新下载或解压。
